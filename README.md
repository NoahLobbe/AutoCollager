# AutoCollager

## Installation
1. Install Python
2. Download this package??
3. Setup virtual environment
   - Windows
     - `> py -m venv .venv`
     - `> cd .venv/Scripts/`
     - `> activate`
     - `(.venv) C:\path\to\directory> pip install requirements.txt`
   - Mac
     - `$ python3 -m venv .venv`
     - `$ source .venv/bin/activate`
     - `(.venv) path/to/directory/$ pip install requirements.txt`
